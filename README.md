## Summary

Edutitute is a student-focused Android application developed to centralize and simplify access to academic information within an educational institute. Built using Kotlin, XML, and modern Android development practices, the app provides real-time academic profiles, personalized timetables, attendance dashboards, multilingual accessibility, and in-app communication tools. The application is designed for scalability, performance, and inclusivity.

---

## Features

- **Real-Time Academic Profiles**
  - Access up-to-date student information, attendance status, and academic records.

- **Personalized Timetables & Attendance Dashboard**
  - High-usage feature set identified through institutional user feedback analytics.

- **Multilingual Support**
  - Supported languages: English, Urdu, Hindi, Kashmiri  
  - Improved accessibility and increased platform engagement by approximately 30% in the first quarter.

- **In-App Messaging System**
  - Subject-specific group channels for academic discussions.
  - Enables seamless peer-to-peer and collaborative communication.
